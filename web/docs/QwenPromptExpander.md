# Qwen Prompt Expander

Local AI prompt generator for ComfyUI using Qwen/SmolLM2 models.

## Quick Start

1. Add the node to your workflow
2. Choose a model (SmolLM2-1.7B recommended)
3. Enter your subject (e.g., "cat in forest")
4. Generate!

## Parameters

- **Model Source**: Choose preset, custom HuggingFace, or local path
- **Precision**: 4-bit (lightest) to fp16 (best quality)
- **Subject**: Your main prompt idea
- **Style/Lighting/Quality**: Optional presets

## First Time Setup

Run the **Qwen Diagnostics** node first to test your system!

## Examples

Input: "cyberpunk girl"
Output: "cyberpunk girl in high-tech environment with sleek lines, sharp angles, digital textures, deep shadows, cinematic lighting effects, 8k"
